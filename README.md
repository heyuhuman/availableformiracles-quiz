# Money Pattern Quiz (Frontend Prototype)

Static HTML/CSS/JS quiz prototype with:
- One-question-per-screen flow
- Back button + progress bar
- DOB “scroll wheel” style selects
- Birthplace auto-detect (geolocation + reverse geocode)
- Animated “Mapping…” interstitial
- Mini “snapshot” preview page
- Opt-in + extra segmentation question
- Formspark submission (includes all answers + score breakdown)

## Run locally
Open `index.html` in your browser.

> Note: geolocation requires HTTPS (or `localhost`).
